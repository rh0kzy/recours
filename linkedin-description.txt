🎓 Système de Gestion des Demandes de Changement de Spécialité - USTHB

J'ai développé une application web complète permettant aux étudiants de l'Université des Sciences et de Technologie Houari Boumediene (USTHB) de soumettre leurs demandes de changement de spécialité en ligne.

🔧 Technologies utilisées :
• Frontend : Next.js 15, React 19, TypeScript, Tailwind CSS
• Backend : Netlify Functions, PostgreSQL (Supabase)
• Infrastructure : Déploiement automatique sur Netlify avec CI/CD

✨ Fonctionnalités clés :
• Interface étudiante responsive avec validation en temps réel
• Panel d'administration pour traiter les demandes
• Transfert automatique de spécialité en base de données
• Système de notifications par email avec templates HTML
• Gestion en masse des demandes avec confirmations

🎯 Impact :
• Digitalisation complète du processus administratif
• Réduction significative du temps de traitement
• Interface intuitive accessible sur tous les appareils
• Automatisation des tâches répétitives

🚀 Projet en production depuis septembre 2025, démontrant mes compétences en développement full-stack moderne et en résolution de problèmes administratifs par la technologie.
